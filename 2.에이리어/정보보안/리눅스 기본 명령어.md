## **리눅스 기본 명령어 (Linux Basic Commands)**

리눅스 커맨드 라인 인터페이스(CLI) 환경에서 자주 사용되는 필수 명령어 모음.

### **명령어 기본 구조 (Command Structure)**

- `명령어 [옵션...] [인자...]` `command [options...] [arguments...]`
    - **명령어 (command):** 실행하고자 하는 프로그램 이름 (예: `ls`, `cp`, `mkdir`).
    - **옵션 (options):** 명령어의 동작 방식을 변경하는 설정. 주로 하이픈(`-`) 또는 더블 하이픈(`--`)으로 시작함 (예: `ls -l`, `ls --all`). 여러 개 사용 가능.
    - **인자 (arguments):** 명령어가 처리할 대상. 주로 파일명이나 디렉토리 경로 등 (예: `cp source.txt target.txt` 에서 `source.txt`와 `target.txt`).

### **1. 탐색 및 경로 관련 명령어 (Navigation & Path)**

- `pwd` (Print Working Directory): 현재 작업 중인 디렉토리의 전체 경로를 출력함.
- `cd [디렉토리 경로]` (Change Directory): 지정한 디렉토리로 작업 위치를 이동함.
    - `cd ~`: 현재 로그인된 사용자의 홈 디렉토리로 이동함.
    - `cd /`: 최상위 루트 디렉토리로 이동함.
    - `cd ..`: 현재 디렉토리의 상위 디렉토리로 이동함.
    - `cd -`: 바로 이전에 있었던 디렉토리로 이동함.
- `ls [옵션...] [경로...]` (List): 지정한 경로의 파일 및 디렉토리 목록을 출력함. 경로 미지정 시 현재 디렉토리 목록 출력.
    - `ls -l`: 파일/디렉토리의 상세 정보(권한, 소유자, 그룹, 크기, 수정일시 등)를 긴 형식(long format)으로 출력함.
    - `ls -a` (all): 이름이 `.`으로 시작하는 숨김 파일 및 디렉토리를 포함한 모든 항목 출력.
    - `ls -h` (human-readable): 파일 크기를 사람이 읽기 쉬운 단위(KB, MB, GB 등)로 표시함 (`-l`과 함께 사용).
    - `ls -t`: 수정된 시간 순서대로 (최신 항목 먼저) 정렬하여 출력함.
    - `ls -r` (reverse): 정렬 순서를 역순으로 출력함.

### **2. 파일 및 디렉토리 관리 명령어 (File & Directory Management)**

- `mkdir [옵션...] 디렉토리명...` (Make Directory): 새로운 디렉토리를 생성함.
    - `mkdir -p 경로/중간디렉토리/새디렉토리`: 경로상의 중간 디렉토리가 없을 경우 함께 생성함 (parents).
- `rmdir [옵션...] 디렉토리명...` (Remove Directory): 비어 있는 디렉토리를 삭제함.
- `cp [옵션...] 원본 대상` (Copy): 파일이나 디렉토리를 복사함.
    - `cp file1 file2`: file1을 file2라는 이름으로 복사 (덮어쓰기 주의).
    - `cp file1 dir1/`: file1을 dir1 디렉토리 안으로 복사.
    - `cp -r dir1 dir2/`: dir1 디렉토리 전체(하위 내용 포함)를 dir2 디렉토리 안으로 복사 (recursive).
    - `cp -i`: 복사 시 대상 파일이 이미 존재하면 덮어쓸지 물어봄 (interactive).
- `mv [옵션...] 원본 대상` (Move): 파일이나 디렉토리의 이름을 변경하거나 다른 위치로 이동시킴.
    - `mv old_name new_name`: 파일/디렉토리 이름을 old_name에서 new_name으로 변경.
    - `mv file1 dir1/`: file1을 dir1 디렉토리 안으로 이동.
    - `mv dir1 dir2/`: dir1 디렉토리를 dir2 디렉토리 안으로 이동 (dir2가 존재할 경우).
    - `mv -i`: 이동/이름 변경 시 대상이 이미 존재하면 덮어쓸지 물어봄 (interactive).
- `rm [옵션...] 파일/디렉토리명...` (Remove): 파일이나 디렉토리를 삭제함. **주의: 휴지통 없이 영구 삭제됨.**
    - `rm file1`: file1 파일을 삭제.
    - `rm -r dir1`: dir1 디렉토리와 그 안의 모든 내용을 재귀적으로 삭제 (recursive).
    - `rm -f file1`: 삭제 확인 질문 없이 강제로 삭제 (force). **매우 주의해서 사용해야 함.**
    - `rm -i file1`: 삭제 전 확인 질문을 함 (interactive).
- `touch [옵션...] 파일명...`: 빈 파일을 생성하거나, 이미 파일이 존재하면 최종 수정 시간을 현재 시간으로 갱신함.

### **3. 파일 내용 확인 명령어 (Viewing File Contents)**

- `cat [옵션...] 파일명...` (Concatenate): 하나 이상의 파일 내용을 순서대로 연결하여 화면(표준 출력)에 전부 출력함. 짧은 파일 확인에 유용.
- `less [옵션...] 파일명...`: 파일 내용을 페이지 단위로 보여주는 뷰어. 대용량 파일 확인에 적합.
    - Space bar: 다음 페이지 이동.
    - `b`: 이전 페이지 이동.
    - `/검색어`: 파일 내에서 검색.
    - `n`: 다음 검색 결과로 이동.
    - `N`: 이전 검색 결과로 이동.
    - `q`: 종료.
- `head [옵션...] 파일명...`: 파일의 앞부분 몇 줄을 출력함 (기본 10줄).
    - `head -n 20 file.txt`: file.txt의 앞부분 20줄 출력.
- `tail [옵션...] 파일명...`: 파일의 뒷부분 몇 줄을 출력함 (기본 10줄).
    - `tail -n 20 file.txt`: file.txt의 뒷부분 20줄 출력.
    - `tail -f file.log` (follow): file.log 파일에 추가되는 내용을 실시간으로 계속 출력함. 로그 파일 모니터링에 매우 유용 (`Ctrl+C`로 종료).

### **4. 검색 관련 명령어 (Searching)**

- `grep [옵션...] 패턴 [파일명...]` (Global Regular Expression Print): 파일 내용이나 표준 입력에서 특정 문자열 패턴을 포함하는 라인을 찾아 출력함.
    - `grep "error" log.txt`: log.txt 파일에서 "error" 문자열이 포함된 라인 검색.
    - `grep -i "error" log.txt` (ignore case): 대소문자 구분 없이 검색.
    - `grep -r "pattern" /path/to/dir` (recursive): 지정된 디렉토리 및 하위 디렉토리 내 모든 파일에서 패턴 검색.
    - `grep -v "pattern" file.txt` (invert match): 패턴과 일치하지 않는 라인 출력.
- `find [경로...] [조건...] [동작...]`: 지정한 경로 아래에서 특정 조건에 맞는 파일이나 디렉토리를 검색함. 매우 강력하고 다양한 옵션 존재.
    - `find /home -name "*.log"`: /home 디렉토리 아래에서 이름이 `.log`로 끝나는 파일 검색.
    - `find /etc -type d`: /etc 디렉토리 아래에서 타입이 디렉토리인 것만 검색 (`-type f`: 파일).
    - `find /var -mtime -7`: /var 디렉토리 아래에서 최근 7일 이내에 수정된 파일/디렉토리 검색.
    - `find . -maxdepth 1 -name "config*"`: 현재 디렉토리(`.`)에서 깊이 1까지만(바로 아래) 이름이 "config"로 시작하는 것 검색.

### **5. 입출력 리디렉션 및 파이프 (I/O Redirection & Pipes)**
- `>` : 명령어 출력을 파일로 저장 (덮어쓰기). 예: `ls -l > list.txt`
- `>>` : 명령어 출력을 파일 끝에 추가. 예: `echo "log" >> events.log`
- `<` : 파일 내용을 명령어 입력으로 사용. 예: `sort < data.txt`
- `|` (파이프): 한 명령어의 출력을 다른 명령어의 입력으로 연결. 예: `ps aux | grep "nginx"`

### **6. 기타 유용한 명령어 (Other Useful Commands)**

- `history`: 현재 셸 세션에서 사용했던 명령어 기록을 보여줌.
- `exit`: 현재 셸 세션을 종료함 (로그아웃 또는 터미널 창 닫기).
- `clear`: 터미널 화면을 깨끗하게 지움.
- `man 명령어` (Manual): 해당 명령어의 공식 매뉴얼 페이지를 보여줌. (`q`로 종료).
- `명령어 --help`: 많은 명령어들이 제공하는 간단한 사용법 및 옵션 목록 출력. `man`보다 가볍게 확인 가능.
- `su [옵션...] [사용자명]` (Switch User): 다른 사용자로 전환함. (이전 문서 '사용자' 부분 참고)
- `sudo 명령어` (Superuser Do): 현재 사용자로 로그인한 상태에서 일시적으로 root 권한을 얻어 명령어를 실행함 (sudoers 설정 필요).


*각 명령어의 더 자세한 옵션과 사용법은 `man` 페이지나 `--help` 옵션을 통해 확인하는 것이 좋음.*

| ---------- | -------------------- | -------------------------------------------------- |
| 탐색 및 경로    | `pwd`                  | 현재 작업 디렉토리 경로 출력                                   |
| 탐색 및 경로    | `cd [경로]`              | 지정한 디렉토리로 이동 (~: 홈, ..: 상위, -: 이전)                 |
| 탐색 및 경로    | `ls [옵션] [경로]`         | 디렉토리 내용 목록 출력 (-l: 상세, -a: 숨김 파일 포함)               |
| 파일/디렉토리 관리 | `mkdir [옵션] <이름>`      | 디렉토리 생성 (-p: 경로상 부모 디렉토리 함께 생성)                    |
| 파일/디렉토리 관리 | `rmdir <이름>`           | 빈 디렉토리 삭제                                          |
| 파일/디렉토리 관리 | `cp [옵션] <원본> <대상>`    | 파일/디렉토리 복사 (-r: 디렉토리 포함)                           |
| 파일/디렉토리 관리 | `mv <원본> <대상>`         | 파일/디렉토리 이동 또는 이름 변경                                |
| 파일/디렉토리 관리 | `rm [옵션] <대상>`         | 파일/디렉토리 삭제 (-r: 디렉토리 포함, -f: 강제 삭제 - 주의)           |
| 파일/디렉토리 관리 | `touch <파일명>`          | 빈 파일 생성 또는 기존 파일 날짜 정보 갱신                          |
| 파일 내용 확인   | `cat <파일명>`            | 파일 내용 전체를 화면에 출력                                   |
| 파일 내용 확인   | `less <파일명>`           | 파일 내용을 페이지 단위로 확인 (대용량 파일용, q로 종료)                 |
| 파일 내용 확인   | `head [옵션] <파일명>`      | 파일 앞부분 출력 (기본 10줄, -n 숫자: 줄 수 지정)                  |
| 파일 내용 확인   | `tail [옵션] <파일명>`      | 파일 뒷부분 출력 (기본 10줄, -n 숫자: 줄 수 지정, -f: 실시간 추적)      |
| 검색         | `grep [옵션] <패턴> <파일명>` | 파일 내에서 특정 문자열(패턴) 검색 (-i: 대소문자 무시, -r: 하위 디렉토리 포함) |
| 검색         | `find <경로> [조건]`       | 특정 조건으로 파일/디렉토리 검색 (예: -name "*.txt", -type f)     |
| 사용자/권한     | `su [옵션] [사용자명]`       | 사용자 계정 전환 (su -: root 환경으로 전환)                     |
| 사용자/권한     | `sudo <명령어>`           | 현재 사용자로 로그인한 상태에서 임시 관리자 권한으로 명령어 실행               |
| 기타         | `history`              | 명령어 사용 기록 출력                                       |
| 기타         | `exit`                 | 현재 셸 종료 (로그아웃)                                     |
| 기타         | `clear`                | 터미널 화면 지우기                                         |
| 기타         | `man <명령어>`            | 명령어 매뉴얼 페이지 보기                                     |
| 기타         | `<명령어> --help`         | 명령어 자체의 간단한 도움말 보기                                 |

## 🔗 아웃링크
```dataview
LIST WITHOUT ID outgoing
FROM ""
WHERE file = this.file
FLATTEN file.outlinks as outgoing
SORT outgoing ASC
```
## 🔙 백링크
```dataview
LIST WITHOUT ID file.link
WHERE contains(file.outlinks, this.file.link)
SORT file.link ASC
```
