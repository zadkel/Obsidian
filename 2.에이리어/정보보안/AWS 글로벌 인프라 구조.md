## **AWS 글로벌 인프라스트럭처 (AWS Global Infrastructure)**

### **1. 개요**

- AWS 클라우드 서비스를 전 세계 사용자에게 안정적이고 낮은 지연 시간(Low Latency)으로 제공하기 위한 **물리적인 기반 시설** 전체를 의미함.
- 전 세계 주요 지점에 데이터 센터와 네트워크를 구축하여 **규모의 경제, 고가용성, 내결함성, 확장성**을 제공하는 것이 핵심.

### **2. 리전 (Region)**

![[Pasted image 20250415225847.png]]
- **정의:** AWS가 데이터 센터들을 클러스터 형태로 운영하는 **전 세계의 물리적인 지리적 위치(국가 또는 도시 단위)**. 각 리전은 다른 리전과 완전히 **분리 및 격리**되어 설계됨.
- **목적:**
    - 사용자와 가까운 위치에 리소스를 배치하여 **네트워크 지연 시간 감소**.
    - 각국의 **데이터 상주(Data Residency) 요구 사항** 충족.
    - 다른 리전에 장애 발생 시 영향을 받지 않는 **지역적 장애 격리** 제공.
- **구성:** 각 리전은 **2개 이상의 가용 영역(Availability Zone)** 으로 구성됨.
- **예시:** `ap-northeast-2` (서울 리전), `us-east-1` (미국 버지니아 북부 리전).

### **3. 가용 영역 (AZ - Availability Zone)**

![[Pasted image 20250415225826.png]]
- **정의:** 하나의 리전(Region) 내에 존재하는, **물리적으로 분리된 하나 이상의 데이터 센터** 그룹. 각 AZ는 독립적인 전력, 냉각, 물리적 보안 및 **매우 짧은 지연 시간(Low-latency)의 전용 네트워크**로 서로 연결됨.
- **목적:**
    - 단일 데이터 센터의 장애(정전, 네트워크 문제, 화재 등) 발생 시에도 다른 AZ에서 서비스가 계속 운영될 수 있도록 **고가용성(High Availability) 및 내결함성(Fault Tolerance)** 제공.
- **구성:** 각 리전은 일반적으로 3개 이상의 AZ로 구성됨 (최소 2개 보장). AZ 간 거리는 의미 있는 수준으로 떨어져 있어 동시 장애 가능성 최소화.
- **활용:** 안정적인 애플리케이션 운영을 위해 **여러 AZ에 걸쳐 리소스(예: EC2 인스턴스, 데이터베이스)를 분산 배포**하는 것이 핵심적인 모범 사례임.

### **4. 엣지 로케이션 (Edge Location)**

- **정의:** AWS 리전이나 AZ보다 훨씬 **더 많은 숫자로 전 세계 주요 도시에 분산되어 위치**하는 AWS 인프라 거점.
- **목적:**
    - 최종 사용자와 **가장 가까운 위치에서 콘텐츠를 캐싱(Caching)하고 전송**하여 **지연 시간을 최소화**하는 데 주로 사용됨.
    - 주요 사용 서비스: **Amazon CloudFront (CDN)**, **Route 53 (DNS)**, AWS Global Accelerator 등.
- **특징:** EC2 인스턴스나 RDS 데이터베이스 같은 핵심 컴퓨팅/데이터베이스 리소스를 직접 실행하는 곳은 아님 (이들은 리전/AZ에서 실행됨). CDN 캐시 서버나 DNS 서버 등이 위치함.

### **5. 글로벌 백본 네트워크 (Global Backbone Network)**

- **정의:** AWS의 리전, 가용 영역(AZ), 엣지 로케이션 등을 **서로 연결하는 AWS 소유의 광대역 프라이빗 네트워크 인프라**. 완전히 이중화된 100GbE 광 네트워크로 구성 (지속 확장 중).
- **목적:**
    - AWS 서비스 간, 리전 간 데이터 전송 시 **공용 인터넷 의존도를 낮추고, 더 안정적이고 빠르며 보안성이 높은 통신** 보장.
    - 전반적인 AWS 서비스 성능 및 안정성 향상에 기여.

---

요약하면, AWS는 전 세계 **리전**에 기반을 두고, 각 리전 내 **가용 영역**을 통해 고가용성을 확보하며, **엣지 로케이션**으로 사용자에게 더 빠른 콘텐츠 전송을 지원하고, 이 모든 것을 자체 **글로벌 네트워크**로 연결하여 안정적인 클라우드 서비스를 제공함.
